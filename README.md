# 🛡️ VaultGuard — Security Audit Dashboard

Continuous security monitoring and vulnerability management platform with automated remediation workflows.

## Features

- 🛡️ **Vuln Scanner** — Automated CVE scanning for containers, dependencies, and infrastructure with false positive filtering.
- 🔍 **Threat Intelligence** — Real-time threat feed correlation with MITRE ATT&CK mapping and risk scoring.
- ⚙️ **Auto Remediation** — Policy-driven auto-patching for critical vulnerabilities with rollback capabilities.
- 📋 **Compliance Reports** — One-click SOC 2, ISO 27001, and PCI DSS compliance reports with evidence collection.

## Quick Start

```bash
npm install -g vault-guard
vault-guard init my-project
cd my-project
vault-guard dev
```
